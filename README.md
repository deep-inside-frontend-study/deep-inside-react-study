# Inside React Study Group
> ‘다시 깊게 익히는 인사이드 리액트’ 함께 읽고 토론하는 스터디 저장소

## 스터디 안내

이 저장소는 아래 목적으로 운영됩니다:

✔ React의 내부 작동 원리를 **구조 중심으로 이해**  
✔ 책 내용을 **단순 요약이 아닌 재해석/Explain으로 정리**  
✔ 실습 코드 / 의견 / 질문을 모두 함께 공유  

## 스터디 목표
- React의 **렌더링 메커니즘**, **상태 관리**, **Fiber 구조**를 내부 동작 관점에서 이해
- “어떻게 쓰는가”가 아니라 “왜 이렇게 동작하는가”를 설명할 수 있는 수준 도달
- 실무 코드에 적용 가능한 인사이트 정리


## 스터디 진행 방식

1️⃣ 매주 **각자 1~3주 분량**을 정리해요.

2️⃣ 각자 정리한 내용을 **Summary + Q&A + Opinion** 형식으로 작성해요.

3️⃣ 스터디 당일 **뽑기**로 대표 2명이 설명해요.

4️⃣ 서로 질문하고 설명해주며 함께 이해 확장, 단순 요약 ❌ → **이해한 방식으로 재해석 ⭕**



## 저장소 구조

```text
.
├── README.md
├── RULES.md
├── CONTRIBUTING.md
├── weeks/
│   ├── week01/
│     ├── summary.md      # 핵심 내용 요약 (개념 중심)
│     ├── questions.md    # 이해 안 된 점 / 토론 주제
│     └── insights.md     # 실무 적용 인사이트
```
